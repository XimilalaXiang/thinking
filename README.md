# Thinking Skills

A collection of mental model skills for AI coding agents. These skills encode frameworks from the world's best thinkers into reusable, trigger-ready modules that help AI assistants make better decisions.

## Available Skills

| Skill | Description | Source |
|-------|-------------|--------|
| [vanity-metrics](skills/vanity-metrics/) | Evaluate goals and metrics using three frameworks: Goal Path, Ability Spectrum, and Virtue Spectrum | [Julian Shapiro](https://www.julian.com/blog/vanity-metrics) |

## Installation

```bash
npx skills add XimilalaXiang/thinking@vanity-metrics -g -y
```

## What Are Thinking Skills?

Unlike traditional coding skills that help with specific technical tasks, thinking skills provide **mental frameworks** that improve how AI assistants reason about problems, evaluate strategies, and guide users toward better decisions.

These skills are designed to be:
- **Framework-driven**: Based on proven mental models from established thinkers
- **Trigger-ready**: Automatically activate when users discuss relevant topics
- **Actionable**: Provide structured output templates for consistent analysis
